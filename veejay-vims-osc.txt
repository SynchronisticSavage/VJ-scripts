Environment variables:
	SDL_VIDEO_HWACCEL		Set to 1 to use SDL video hardware accel (default=on)
	VEEJAY_PERFORMANCE		Set to "quality" or "fastest" (default is fastest)
	VEEJAY_AUTO_SCALE_PIXELS	Set to 1 to convert between CCIR 601 and JPEG automatically (default=dont care)
	VEEJAY_INTERPOLATE_CHROMA	Set to 1 if you wish to interpolate every chroma sample when scaling (default=0)
	VEEJAY_CAPTURE_DRIVER		Set to "unicap" or "v4lutils" (default=v4lutils)
	VEEJAY_SDL_KEY_REPEAT_INTERVAL	interval of key pressed to repeat while pressed down.
	VEEJAY_PLAYBACK_CACHE		Sample cache size in MB - by default, veejay takes 30 percent of total RAM
	VEEJAY_SDL_KEY_REPEAT_DELAY	Delay key repeat in ms
	VEEJAY_FULLSCREEN		Start in fullscreen (1) or windowed (0) mode
	VEEJAY_SCREEN_GEOMETRY		Specifiy a geometry for veejay to position the video window.
	VEEJAY_SCREEN_SIZE		Size of video window, defaults to full screen size.
	VEEJAY_RUN_MODE			Run in "classic" (352x288 Dummy) or default (720x576). 


	Example for bash:
			$ export VEEJAY_AUTO_SCALE_PIXEL=1
